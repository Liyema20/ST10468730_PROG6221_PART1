# ST10468730_PROG6221_PART1
🔐 Cybersecurity Awareness Chatbot

📌 Overview

The Cybersecurity Awareness Chatbot is a C# console-based program that will inform its users about web safety in an interactive and entertaining manner.

The chatbot is both text-Based and voice-Based, enabling users to get educated about significant cybersecurity issues like phishing, password security and social engineering.

🚀 Features

🔐 User Authentication System

Registration and login of users.
* Sha256 hash password storage.
* “Remember Me” functionality
* Guest access option

🎤 Voice Interaction

* Voice recognition on start up.
* Text-to-speech responses
* Adjustable voice speed
* Multiple voice options:

  * Female voice
  * Male voice
  * Default system voice
  * Custom voice
  * Voice off option.

💬 Interactive Chatbot

* Replies to questions of the user.
* Keyword-based response system
Friendly and organised communication.

📚 Cybersecurity Learning System

Learners are able to choose the subjects and study by:

* Definition
* Real-life Scenario
* Solution

🧠 Smart Learning Flow

* Uploads: queries user whether he/she is aware of the subject.
* Re-explains if needed
* Offers solutions on the feedback of the users.

🛠 Input Validation

Prevents empty or invalid input.
* Provides a good user experience.

🎨 Console UI

* Neat and well-defined interface.
* ASCII art logo
* Color-enhanced output

🏗️ Project Structure

* Program.cs
  Regulates the overall flow and interaction with the user.

* UserManager.cs
  Processes the users registration and logins, logouts and file management.

* ResponseManager.cs
  Supervises chatbot replies and cybersecurity issues.

* InputValidator.cs
  Validates user input

* AsciiArt.cs
  Showcases the logo of the chatbot.

* VoiceGreeting.cs
  Controls voice output and voice settings.

▶️ How to Run

1. Open project in Visual Studio.
2. Build the solution
3. Run the program
4. Do as the instructions on the screen tell you.

🎯 Purpose of the Project

This chatbot is aimed at:
